Brayan Alexander Paredes Sánchez - 20171020106<br/>
Juan Camilo Martínez López - 20171020074</br>

el 21 de octubre de 1994; sus integrantes eran: Erich Gamma, John Vlissides, Ralph Johnson y Richard Helm.
